Projects

Gmap:

The first project is a wrapper of nmap (network mapper) named gmap that one can use as a full-force network mapper. Gmap takes the golden-standard usability nmap has to offer and provides the maximum functionality nmap has to offer, scanning all ports regardless of routing status, returning operating system information and versions. Built in 1997, nmap was ran on a fraction of the computing power we have today. Instead of scanning the usual 100 or 1000 ports nmap scans for resource management, the CLI flags gmap has in the pythons script scan all 65,535 ports and returns all information nmap has to offer from 7 different flags, in one command reducing need for memorization and introducing simplicity.

Chainsaw demonstration:

The third project is a safety demonstration for chainsaws in order to educate one before using a chainsaw in order to prevent workplace accidents. The program is a 3d interactive model of a chainsaw with every part of a standard chainsaw labeled for one to see and take note of, along with demonstrating how to turn it on safely, and make sure it continues to work. A chainsaw is not so straight forward, and it is detrimental that the operator knows the workings of a chainsaw in order to work efficiently. Note: The program is in it's early stages and there are more demonstration features to be implemented.
