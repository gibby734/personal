Projects

Project 1

The first project is a spinoff of nmap (network mapper) named gmap that one can use as a one-stop-shop network mapper. gmap takes the golden-standard usability nmap has to offer and provides the maximum functionality nmap has to offer. Built in 1997, nmap was ran on a fraction of the computing power we have today. Instead of scanning the usual 100 or 1000 ports nmap scans for resource management, the CLI flags gmap has in the pythons script scan all 65,535 ports and returns all information nmap has to offer from 7 different flags, in one command reducing need for memorization and introducing simplicity.
